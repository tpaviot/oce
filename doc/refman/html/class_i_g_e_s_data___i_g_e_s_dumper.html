Detailed Description

Provides a way to obtain a clear Dump of an IGESEntity (distinct from normalized output). It works with tools attached to Entities, as for normalized Reade and Write.

For each Entity, displaying data is splitted in own data (specific to each type) and other attached data, which are defined for all IGES Types (either from "Directory Entry" or from Lists of Associativities and Properties)
