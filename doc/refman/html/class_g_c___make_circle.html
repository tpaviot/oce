Detailed Description

This class implements the following algorithms used to create Cirlec from Geom.

  • Create a Circle parallel to another and passing though a point.
  • Create a Circle parallel to another at the distance Dist.
  • Create a Circle passing through 3 points.
  • Create a Circle with its center and the normal of its plane and its radius.
  • Create a Circle with its axis and radius. The circle's parameter is the angle (Radian). The parametrization range is [0,2*PI]. The circle is a closed and periodic curve. The center of the circle is the Location point of its axis placement. The XDirection of the axis placement defines the origin of the parametrization.
