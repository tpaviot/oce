Detailed Description

Implements construction algorithms for an arc of hyperbola in 3D space. The result is a Geom_TrimmedCurve curve. A MakeArcOfHyperbola object provides a framework for:

  • defining the construction of the arc of hyperbola,
  • implementing the construction algorithm, and
  • consulting the results. In particular, the Value function returns the constructed arc of hyperbola.
