Detailed Description

The abstract class BooleanOperation is the root class of Boolean operations. A BooleanOperation object stores the two shapes in preparation for the Boolean operation specified in one of the classes inheriting from this one. These include:

  • Common
  • Cut
  • Fuse
  • Section.
