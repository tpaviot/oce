Detailed Description

Root class of drivers to build draw variables from TDF_Label. Priority rule to display standard attributes is :

  • 1 Constraint
  • 2 Object
  • 3 Datum (Point,Axis,Plane)
  • 4 Geometry
  • 5 NamedShape
