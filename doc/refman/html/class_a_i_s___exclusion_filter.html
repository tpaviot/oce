Detailed Description

A framework to reject or to accept only objects of given types and/or signatures. Objects are stored, and the stored objects - along with the flag settings - are used to define the filter. Objects to be filtered are compared with the stored objects added to the filter, and are accepted or rejected according to the exclusion flag setting.

  • Exclusion flag on
  • the function IsOk answers true for all objects, except those of the types and signatures stored in the filter framework
  • Exclusion flag off
  • the funciton IsOk answers true for all objects which have the same type and signature as the stored ones.
